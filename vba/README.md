# VBA modules

このフォルダには、ClinicAppointmentで使用するExcel VBAモジュールを格納します。

## 現在のモジュール

- `AppointmentBook_Phase1.bas`

## インポート方法

1. Excelを `.xlsm` 形式で保存する。
2. `Alt + F11` でVBAエディタを開く。
3. 古いモジュールがある場合は右クリックして削除する。
4. `ファイル` → `ファイルのインポート` から `.bas` ファイルを選択する。
5. `Alt + F8` で `GenerateAppointmentBook_Phase4` を実行する。

## 必要シート

- `Template`
- `Settings`
- `Output`

## Settings

- `B2`: 年
- `B3`: 月
- `B5:F5`: 列見出し。
- `B7:F13`: 曜日別の勤務パターン。
- `A7:A13`: 人が確認しやすくするための曜日ラベル。マクロ処理には使用しない。

列見出しの対応は以下の通り。

- `B5` → Output列B：1列目Dr
- `C5` → Output列D：2列目Dr
- `D5` → Output列F：予備枠
- `E5` → Output列I：1列目DH
- `F5` → Output列J：2列目DH

TemplateのH列は意図的に幅が狭いスペーサー列として扱うため、担当者名の出力先には使用しない。

`B5:F5` がすべて空欄の場合は、Templateに入っている列見出しをそのまま使用する。

## 曜日別勤務パターン

`B7:F13` は、左から列見出しと同じ順序で、上から月曜〜日曜として扱う。
曜日名は `A7:A13` に入力しておくと見やすいが、マクロは `A7:A13` を読み取らない。

```text
      A     B     C     D     E     F
5           Dr1   Dr2   予備  DH1   DH2
7     月
8     火
9     水
10    木
11    金
12    土
13    日
```

つまり、勤務パターンの入力欄は以下のように使う。

```text
      A     B     C     D     E     F
5           Dr1   Dr2   予備  DH1   DH2
7     月
8     火
9     水
10    木
11    金
12    土                         休
13    日
```

上の例では、土曜日の2列目DHだけが休み扱いになる。

空欄、`出勤`、`勤務`、`on` などは出勤扱い。以下の値は休み扱いとして、その担当者欄を網掛けし、見出しに `休` を追記する。

```text
休 / 休み / 休診 / off / x / × / 0 / - / 欠
```

`B7:F13` がすべて空欄の場合は、担当者ごとの勤務パターン処理は行わない。
