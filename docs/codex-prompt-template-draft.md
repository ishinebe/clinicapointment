# Codex依頼プロンプト: TemplateDraft作成フェーズ

## 背景

ClinicAppointment は、紙運用前提のクリニック用アポ帳を Excel VBA で自動生成するプロジェクトです。

現在の設計方針は以下です。

- `Template` は現行アポ帳の保存用デザインマスターとして扱う。
- `Output` はTemplateをもとに生成する印刷用シート。
- 現行アポ帳のDr列、予備枠、DH列の列幅・配置・罫線・フォントはできるだけ踏襲する。
- 一方で、現行Templateの時間表示は旧様式で視認性が低く、セル結合や時刻分割が含まれている可能性がある。
- したがって、時間列周辺はTemplateをそのまま踏襲せず、マクロ側で再構成してよい。
- Template本体を直接壊さないため、まず `TemplateDraft` シートを作成し、そこで改良版テンプレートを試作する。

## 重要な修正方針

初回の `TemplateDraft` では、Templateのセル結合をそのまま残したため、時間列が旧様式に引っ張られた。
今後は、**時間列および時間列に隣接する時刻補助列のセル結合は解除してよい**。

特に以下を重視する。

- Dr列、予備枠、DH列の幅や大枠はTemplateを踏襲する。
- 時間列はTemplateを正としない。
- 時間列はマクロで `9:00`, `9:15`, `9:30`, `9:45` の単一列表示へ再構成する。
- 時間表示のために無理にセル結合を維持しない。
- 旧Templateにある「9」「10」列と「00」「15」「30」「45」列の分割構造は破棄してよい。

## 依頼内容

Excel VBAで `CreateTemplateDraft` マクロを改善してください。

## 追加・修正してほしいファイル

- `vba/AppointmentBook_Phase1.bas`

必要に応じて関数を追加・分割してください。

## 実装要件

### 1. TemplateDraftシートの作成

- `Template` シートをコピーして `TemplateDraft` シートを作成する。
- 既に `TemplateDraft` が存在する場合は、確認メッセージを出した上で削除・再作成する。
- `Template` シート本体は絶対に変更しない。

### 2. レイアウト維持の範囲

`TemplateDraft` では、以下をTemplateから踏襲してください。

- Dr列の列幅
- 予備枠の列幅
- DH列の列幅
- Dr列、予備枠、DH列の大枠罫線
- 青点線など、予約欄内部の視認性に関わる補助線
- フォント
- 行高
- 印刷設定

ただし、以下は維持しなくてよい。

- 時間列のセル結合
- 時間列に隣接する旧時刻補助列のセル結合
- 「9」「10」「11」と「00」「15」「30」「45」が分かれた旧時刻表示構造

### 3. 時間列の改善

`TemplateDraft` では、時間表示を以下のように15分刻みで見やすくしてください。

```text
9:00
9:15
9:30
9:45
10:00
10:15
...
```

要件：

- 開始時刻は 9:00。
- 終了表示は少なくとも 19:00 まで対応する。
- 15分刻み。
- 時間表示は単一列にまとめる。
- 00分は太字にする。
- 00分の横罫線は実線または太実線で視認性を高める。
- 15分・30分・45分は薄い点線にする。
- 時間列以外のDr列、予備枠、DH列の幅や配置はTemplateを可能な範囲で踏襲する。

### 4. 時間列の推奨処理

現行Templateの時間周辺構造に依存しすぎないこと。

推奨：

- 旧時刻表示がある列を検出するか、定数で時間列範囲を指定する。
- 時間列周辺の結合を解除する。
- 左端の1列を新しい時間列として使う。
- 旧分表示列が残る場合は、空欄化するか、必要に応じて幅を狭くする。
- 予約欄本体の列幅は維持する。

### 5. 時間外網掛けの試作

`TemplateDraft` 上で、視認性確認用に時間外網掛けを適用できるようにしてください。

初期値：

- TemplateDraftは特定曜日ではなく「1日分テンプレート」なので、コード内定数として `18:00` を仮の診療終了時刻にする。
- `18:00` 以降を薄い網掛けにする。
- 網掛けはボールペンで記入できる薄さにする。

### 6. 実行マクロ

以下のマクロを用意してください。

```vb
Public Sub CreateTemplateDraft()
```

既存のPhase1マクロは残してください。

### 7. 安全性

- `Application.ScreenUpdating`, `Application.DisplayAlerts`, `Application.EnableEvents` は適切に制御し、エラー時にも復元してください。
- `Option Explicit` を維持してください。
- 既存の `GenerateAppointmentBook_Phase1` を壊さないでください。
- Template本体は変更しないでください。

## 完了条件

- `CreateTemplateDraft` を実行すると `TemplateDraft` シートが作成される。
- Template本体は変更されない。
- Dr列、予備枠、DH列の大枠と見た目はTemplateに近い。
- 時間列が `9:00`, `9:15`, `9:30`, `9:45` 形式の単一列表示になる。
- 時間列周辺の不要なセル結合が解除されている。
- 00分と15分刻みの罫線が視認性よく設定される。
- 薄い網掛けが適用できる。

## 補足

現時点では、完璧な最終Output生成よりも、TemplateDraft上で見た目を検証できることを優先してください。
セル結合は、現行ファイル作成時のExcel操作由来である可能性が高いため、時間列については無理に維持しないでください。
