# 開発フェーズ計画

## 現在の重要方針

`Template` は現行アポ帳の保存用デザインマスターとして扱う。
ただし、現行Templateは時間表示が旧様式であるため、直接Output生成に使うだけでは改善後の様式にならない。

そのため、次の中間シートを導入する。

```text
Template      = 現行様式の保存用デザインマスター
TemplateDraft = 改良版テンプレート試作用シート
Output        = 最終印刷用出力
```

まず `Template` を壊さず、`TemplateDraft` にコピーした上で時間列・罫線・時間外網掛けを試作する。
見た目が確定したら、`TemplateDraft` を正式な `Template` として採用する。

## Phase 1.5: TemplateDraft作成

現行Templateを壊さずに、改良版テンプレートを試作するためのフェーズ。

### 実装対象

- `CreateTemplateDraft` マクロを追加する。
- `Template` シートを `TemplateDraft` シートへコピーする。
- 既に `TemplateDraft` が存在する場合は、確認後に削除または上書きする。
- `TemplateDraft` のDr列、予備枠、DH列の列幅・配置・セル結合・フォントはTemplateを踏襲する。
- 時間列のみ、旧様式から `9:00`, `9:15`, `9:30`, `9:45` のような15分刻み表記へ置換する。
- 00分は実線、15分・30分・45分は薄い点線とする。
- 時間外の薄い網掛けは、まず視認性確認用として任意の診療終了時刻を基準に適用できるようにする。
- 生成後、ユーザーがTemplateDraftを目視確認できるようにする。

### 完了条件

- `TemplateDraft` が作成される。
- 現行TemplateのDr列、予備枠、DH列の見た目が保たれている。
- 時間列だけが見やすい15分刻み形式に変わっている。
- Template本体は変更されない。

## Phase 2: 生成結果の実用書式への調整

日付・曜日・時間表示・担当者名だけをマクロで置換する。

### 実装対象

- 日付アンカーセルまたは日付候補セルの明確化
- 時間列の `9:00`, `9:15`, `9:30`, `9:45` 形式への置換
- Settingsに基づく列見出しの置換
- TemplateDraftで確定した時間列仕様をOutput生成処理へ反映する
- 現行アポ帳の「時間列以外」の見た目を可能な限りそのまま踏襲する
