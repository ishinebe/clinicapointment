# 開発フェーズ計画

## Phase 1

TemplateをデザインマスターとしてOutputへコピーし、現行ファイルの見た目を維持する。

### 完了条件

- `Template` シートの1日分デザインを `Output` にコピーできる。
- `Settings!B2` の年、`Settings!B3` の月を読み取り、その月の日数分だけコピーできる。
- 列幅、行高、セル結合、罫線、フォント、印刷設定を可能な限り維持する。
- 日付セルが検出できる場合は日付・曜日を置換する。

## Phase 2

日付・曜日・時間表示・担当者名だけをマクロで置換する。

### 実装対象

- 日付アンカーセルまたは日付候補セルの明確化
- 時間列の `9:00`, `9:15`, `9:30`, `9:45` 形式への置換
- Settingsに基づく列見出しの置換

## Phase 3

曜日別診療時間に基づき、時間外網掛けと休診表示を反映する。

### 実装対象

- 月火金：19:00以降を薄い網掛け
- 水：18:00以降を薄い網掛け
- 土：17:30以降を薄い網掛け
- 木日：休診表示

## Phase 4

Settingsの担当者マスター、列ごとの担当者選択、勤務パターンを反映する。

## Phase 5

Exceptions、祝日、臨時休診、PDF出力、集計機能を追加する。
