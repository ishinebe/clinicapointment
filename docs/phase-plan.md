# 開発フェーズ計画

## 現在の重要方針

`Template` は、現行アポ帳の書式を反映した確定済みデザインマスターとして扱う。

```text
Template = 確定済みの1日分デザインマスター
Output   = 最終印刷用出力
```

現行アポ帳に合わせるための見た目調整は、テンプレート作成作業時に完了済み。
そのため、以後のマクロ実装では `Template!A1:J46` を正解デザインとして扱い、日付など必要最小限の値だけを差し替える。

現在はUI改善フェーズとして、`Settings` シートを受付・歯科助手向けの操作画面として整備する。既存生成ロジックとの互換性維持のため、`Settings!B2`、`Settings!B3`、`Settings!B5:F5`、`Settings!B7:F13`、`Settings!B16:F16`、`Settings!H5:H24` は固定セルとして残す。

---

## 進捗ログ

### 2026-07-07

- `Template` の1日分範囲を `A1:J46` として固定した。
- `UsedRange` による過大コピーを避けるため、1日分テンプレート範囲は固定範囲で取得する方針に変更した。
- 現行アポ帳に合わせるための書式調整は、テンプレート作成作業時に完了済み。
- `Template` をもとに、指定年月の1か月分を `Output` に生成できることを確認済み。
- 確定済みTemplateの時間列をマクロで再描画しない方針に変更した。
- 時間外網掛けは、A列の実際の時刻表示を読み取って判定する方式に変更した。
- 曜日別の時間外網掛けと休診表示の動作確認が完了した。
- Phase 4-A/Bとして、Settings!B5:F5 から列見出しを読み取り、各日ブロックへ反映する処理を追加した。
- Settings UI改善フェーズとして、`Settings` シートを「アポ帳作成 設定画面」に整備する方針へ進んだ。
- `SetupSettingsDropdowns` / `SetupUserFriendlySettings` で、説明文、入力セルの色分け、プルダウン、作成ボタンを配置する。
- `Template` は確定済みデザインマスター、`Output` は生成結果として扱い、Settings UI改善では原則として見た目や生成ロジックを変更しない。

## Phase 1: Templateから1か月分を生成

TemplateをデザインマスターとしてOutputへコピーし、現行ファイルの見た目を維持する。

### 完了条件

- [x] `Template` シートの1日分デザインを `Output` にコピーできる。
- [x] `Settings!B2` の年、`Settings!B3` の月を読み取り、その月の日数分だけコピーできる。
- [x] 列幅、行高、セル結合、罫線、フォント、印刷設定を可能な限り維持する。
- [x] 日付セルが検出できる場合は日付・曜日を置換する。
- [x] `Template!A1:J46` を1日分として固定し、不要なUsedRange拡張による過大生成を防止する。
- [x] 時間列以外の見た目は、テンプレート側で現行アポ帳に合わせて調整済み。

### 現状の確認

Phase 1は完了。
`Template!A1:J46` を1日分として、指定月の日数分だけ `Output` に縦方向へ生成できる。

生成結果の基本書式は、確定済みTemplateをそのまま踏襲する。
今後はTemplateの見た目を大きく変更するのではなく、必要な値の差し替え、曜日別処理、担当者・例外設定の反映を中心に進める。

## Phase 6: Settings UI改善

受付・歯科助手が直接操作する前提で、`Settings` シートを現場向け操作画面として整える。

### 実装対象

- [x] 大きなタイトル「アポ帳作成 設定画面」を配置する。
- [x] 年月、担当者、曜日別勤務パターン、医院全体の当月終了時刻、スタッフ一覧、作成ボタンのセクションを用意する。
- [x] 入力セルに薄い色を付け、見出し・説明文・罫線・列幅を整える。
- [x] `B5:F5`、`B7:F13`、`B16:F16` の既存プルダウンを維持する。
- [x] `H5:H24` を担当者マスターとして分かりやすく表示する。
- [x] `GenerateAppointmentBook_Phase5` を実行する「アポ帳を作成」ボタンを配置する。
- [x] `SetupExceptionsDateDropdowns` を実行する「例外日付を更新」ボタンを配置する。
- [x] セットアップを複数回実行してもボタンが重複しないようにする。

### 通常の使用手順

1. `SetupUserFriendlySettings` または `SetupSettingsDropdowns` を実行する。
2. 年月を設定する。
3. 担当者を選択する。
4. 必要なら曜日別勤務パターンを設定する。
5. 必要なら `Exceptions` を入力する。
6. `Settings` の「アポ帳を作成」ボタンを押す。
