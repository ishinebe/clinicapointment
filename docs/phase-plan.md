# 開発フェーズ計画

## 現在の重要方針

`Template` は、現行アポ帳の書式を反映した確定済みデザインマスターとして扱う。

```text
Template = 確定済みの1日分デザインマスター
Output   = 最終印刷用出力
```

現行アポ帳に合わせるための見た目調整は、テンプレート作成作業時に完了済み。
そのため、以後のマクロ実装では `Template!A1:J46` を正解デザインとして扱い、日付など必要最小限の値だけを差し替える。

---

## 進捗ログ

### 2026-07-07

- `Template` の1日分範囲を `A1:J46` として固定した。
- `UsedRange` による過大コピーを避けるため、1日分テンプレート範囲は固定範囲で取得する方針に変更した。
- 現行アポ帳に合わせるための書式調整は、テンプレート作成作業時に完了済み。
- `Template` をもとに、指定年月の1か月分を `Output` に生成できることを確認済み。
- 現時点では「確定済み1日分テンプレート × 1か月分生成」まで完了。

---

## Phase 1: Templateから1か月分を生成

TemplateをデザインマスターとしてOutputへコピーし、現行ファイルの見た目を維持する。

### 完了条件

- [x] `Template` シートの1日分デザインを `Output` にコピーできる。
- [x] `Settings!B2` の年、`Settings!B3` の月を読み取り、その月の日数分だけコピーできる。
- [x] 列幅、行高、セル結合、罫線、フォント、印刷設定を可能な限り維持する。
- [x] 日付セルが検出できる場合は日付・曜日を置換する。
- [x] `Template!A1:J46` を1日分として固定し、不要なUsedRange拡張による過大生成を防止する。
- [x] 時間列以外の見た目は、テンプレート側で現行アポ帳に合わせて調整済み。

### 現状の確認

Phase 1は完了。
`Template!A1:J46` を1日分として、指定月の日数分だけ `Output` に縦方向へ生成できる。

生成結果の基本書式は、確定済みTemplateをそのまま踏襲する。
今後はTemplateの見た目を大きく変更するのではなく、必要な値の差し替え、曜日別処理、担当者・例外設定の反映を中心に進める。

## Phase 2: 値の差し替えと生成精度の安定化

確定済みTemplateを崩さず、日付・曜日・時間表示・担当者名など、運用上必要な値だけをマクロで置換する。

### 実装対象

- 日付アンカーセルまたは日付候補セルの明確化
- 時間列の `9:00`, `9:15`, `9:30`, `9:45` 形式の維持・必要時のみ置換
- Settingsに基づく列見出しの置換
- 確定済みTemplateの書式を崩さずにOutput生成へ反映する
- 生成後の印刷範囲・改ページ・ページ数の確認を安定化する
